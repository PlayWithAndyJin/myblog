# Windows 无 CUDA 版本安装 TensorFlow

适用于没有 NVIDIA 显卡或不需要 GPU 加速的用户。

## 安装步骤

1. 安装 Python（建议 3.8~3.11）。
2. 打开命令行，执行：

```bash
pip install tensorflow
```
