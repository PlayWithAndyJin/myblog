# 安装依赖

建议使用国内源加速（如淘宝镜像）：

```bash
npm install --registry=https://registry.npmmirror.com
```

:::tip
推荐访问 [我收录的镜像库](http://andyjin.website/tools) 获取最新国内镜像源和包管理配置方法。
:::

如无网络问题，直接：

```bash
npm install
```

如遇到依赖安装慢或报错，请检查 Node.js 和 npm 版本，或切换网络环境。
